## 🔍 분석 단계별 설명

### 01 – 구별 방문객 비율 분석 (`01_구별방문객.ipynb`)

현지인 + 외지인 방문 수 기준 **서울 상위 15개 구**를 선정하고,
각 구의 **외국인 방문 비율 (외국인 / 현지인+외지인)** 을 계산합니다.

- 비율 상위 3구 / 하위 3구 추출 비교
- 현지인 편중 지수(Hidden Gems) 산출 → "현지인만 아는 명소" 후보 도출
- 2×2 차트 시각화

---

### 02 – 지하철 시간대별 외국인 승하차 분석 (`02_지하철_시간대.ipynb`)

외국인 관광객 기간권 데이터를 시간대별로 분류하여 **승/하차 패턴**을 파악합니다.

| 구간 | 시간대 |
|---|---|
| 새벽 | 0 ~ 5시 |
| 오전 | 6 ~ 11시 |
| 오후 | 12 ~ 17시 |
| 야간 | 18 ~ 23시 |

- 외국인 승차 / 하차 TOP 20 역 히트맵
- 자치구별 역 시간대 승차 히트맵
- 인천공항 출발 기준 역별 최소 환승 경로 분석

---

### 03 – 따릉이 대여소 GeoMap (`03_따릉이_대여소.ipynb`)

따릉이 대여소 좌표 데이터를 서울시 GeoJSON과 결합해 **구별 대여소 밀도**를 지도로 시각화합니다.

- Spatial Join으로 대여소 → 자치구 매핑
- Choropleth 히트맵 + 대여소 위치 산점도 오버레이
- 구별 순위 패널 표시

---

### 04 – 외국인 따릉이 이용 현황 (`04_외국인_따릉이_이용률.ipynb`)

2024년 / 2025년 전체 이용량과 외국인 이용량을 비교합니다.

- 외국인 대여 / 반납건수 히트맵
- 전체 vs 외국인 대여량 비교 (로그 스케일)
- 외국인 이용 비율 및 증가율 시각화

---

### 05 – K-Means 군집 분석 (`05_kmeans.ipynb`)

6개 변수를 기반으로 서울 **25개 자치구를 4개 유형**으로 군집화합니다.

**사용 변수:** 외국인 방문 비율, 외국인 방문객 수, 따릉이 정류장 수, 단기체류 외국인 생활인구, 외국인 따릉이 대여건수, 외국인 따릉이 반납건수

| 군집 | 설명 |
|---|---|
| 관광 중심형 | 외국인 비율·방문량 모두 높음 |
| 관광-이동 중간형 | 외국인 유입과 따릉이 이동이 활발 |
| 생활권 잠재형 | 현지인 중심이나 외국인 유입 가능성 있음 |
| 저방문 보완형 | 외국인 방문이 적은 지역 |
