# 🗺️ 서울시 외국인 관광 데이터 분석 (공공데이터 허브 공모전)

서울시 공공데이터를 활용해 **외국인 관광객의 이동 패턴**을 분석하고 시각화한 프로젝트입니다.

---

## 📁 프로젝트 구조

```
1_da.ipynb          # 전체 분석 코드 (Jupyter Notebook)
```

---

## 📊 분석 개요

### 사용 데이터

| 데이터 | 출처 |
|---|---|
| 현지인 / 외지인 / 외국인 지역별 방문자 수 | 공공데이터 허브 |
| 서울교통공사 외국인 관광객 기간권 일별 통행 통계 | 서울 열린데이터 광장 |
| 서울시 공공자전거(따릉이) 대여소 마스터 정보 | 서울 열린데이터 광장 |
| 서울특별시 공공자전거 외국인 대여정보(월별/일별) | 서울 열린데이터 광장 |
| 자치구 단위 서울 생활인구 (단기체류 외국인) | 서울 열린데이터 광장 |

---

## 🔍 분석 단계별 설명

### 1단계 – 구별 방문객 비율 분석

현지인 + 외지인 방문 수 기준 **서울 상위 15개 구**를 선정하고,  
각 구의 **외국인 방문 비율 (외국인 / 현지인+외지인)** 을 계산합니다.

- 비율 상위 3구 / 하위 3구를 추출해 비교
- 2×2 차트로 시각화 (누적 바차트, 비율 바차트, 상위/하위 그룹 비교)

**출력 파일:** `top15_ratio_analysis.png`, `result_top15_ratio.csv`

---

### 2단계 – 현지인 편중 지수 (Hidden Gems 분석)

외국인 대비 현지인 방문 비율이 높은 구를 찾아 **"현지인만 아는 명소"** 후보를 도출합니다.

- 현지인 편중도 = 현지인 수 / (외국인 수 + 1)
- 현지인 방문량이 중위값 이상인 구만 대상
- 산점도(현지인 vs 외국인) + 편중도 Top 10 바차트로 시각화

**출력 파일:** `local_hidden_gems.png`

---

### 3단계 – 지하철 시간대별 외국인 승하차 분석

외국인 관광객 기간권 데이터를 시간대별로 분류하여 **승/하차 패턴**을 파악합니다.

| 구간 | 시간대 |
|---|---|
| 새벽 | 0 ~ 5시 |
| 오전 | 6 ~ 11시 |
| 오후 | 12 ~ 17시 |
| 야간 | 18 ~ 23시 |

- 전체 시간대별 승하차 바차트
- 외국인 승차 TOP 20 역 히트맵
- 외국인 하차 TOP 20 역 히트맵
- 자치구별 역 시간대 승차 히트맵 (종로구, 중구, 용산구, 관악구, 노원구, 강동구)

---

### 4단계 – 따릉이 대여소 GeoMap

따릉이 대여소 좌표 데이터를 서울시 GeoJSON과 결합해 **구별 대여소 밀도**를 지도로 시각화합니다.

- Spatial Join으로 대여소 → 자치구 매핑
- Choropleth 히트맵 + 대여소 위치 산점도 오버레이
- 우측 패널에 구별 순위 표시

**출력 파일:** `ddareungi_geomap.png`, `result_ddareungi_by_gu.csv`

---

### 5단계 – 외국인 따릉이 이용 현황 (2024 vs 2025)

2024년/2025년 전체 이용량과 외국인 이용량을 비교합니다.

- 외국인 이용 비율 바차트 (Plotly)
- 전체 vs 외국인 대여량 비교 (로그 스케일)
- 외국인 이용 증가율 시각화

**출력 파일:** `foreign_bike_ratio.png`, `foreign_vs_total_logscale.png`

---

### 6단계 – K-Means 군집 분석 (25개 자치구)

6개 변수를 기반으로 서울 25개 자치구를 4개 유형으로 군집화합니다.

**사용 변수:**
- 외국인 방문 비율
- 외국인 방문객 수
- 따릉이 정류장 수
- 단기체류 외국인 생활인구
- 외국인 따릉이 대여건수
- 외국인 따릉이 반납건수

**군집 유형:**

| 군집 | 설명 |
|---|---|
| 관광 중심형 | 외국인 비율·방문량 모두 높음 |
| 관광-이동 중간형 | 외국인 유입과 따릉이 이동이 활발 |
| 생활권 잠재형 | 현지인 중심이나 외국인 유입 가능성 있음 |
| 저방문 보완형 | 외국인 방문이 적은 지역 |

결과를 서울시 지도 위에 색상으로 시각화합니다.

**출력 파일:** `kmeans_input_features_25gu.csv`, `kmeans_cluster_result_25gu.csv`, `kmeans_cluster_profile_25gu.csv`
